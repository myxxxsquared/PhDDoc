
# 北京大学博士学位研究生学科综合考试报告Tex参考模板

**本模板仅供参考，不保证与要求模板一致**

北京大学博士学位研究生学科综合考试报告模板

## 字体

由于Windows字体版权限制，请使用copyfont.bat将Windows字体拷贝到font文件夹下，以便于跨平台编译。

## 编译

使用XeLaTeX进行编译
```sh
latexmk -xelatex
```
